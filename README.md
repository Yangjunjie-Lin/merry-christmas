# Animated Christmas Tree

Simple Python script that asks your name and shows an animated, flashing-point Christmas tree in the terminal.

Prerequisites
- Python 3.7+

Install
Open PowerShell and run:

```powershell
python -m pip install -r requirements.txt
```

Run

```powershell
python tree.py
```

Notes
- On Windows, recent terminals (Windows Terminal, PowerShell) support ANSI escape sequences. If colors do not appear, try using Windows Terminal or enable VT processing.
